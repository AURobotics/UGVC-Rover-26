# Console

This is the Console sub-project folder, containing the source code for the remote-control GUI.

## Contributing

### Project Setup

After cloning the mono-repo, open the `rover_ws/` folder inside your favorite text editor or IDE.

### Pre-requisites

#### Installing Pixi [↗️](https://pixi.prefix.dev/latest/installation/)

Windows
```pwsh
winget install prefix-dev.pixi
```

Linux/ MacOS
```sh
curl -fsSL https://pixi.sh/install.sh | bash
```

#### Installing project dependencies

Make sure you are in a pixi-activated shell
```sh
pixi shell
```

Then, run:
```sh
pixi install
```
